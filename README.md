# Projects

## 1. Semantic Search Engine

Built a semantic search engine that:

- Extracts text from PDF documents
- Chunks long documents
- Generates embeddings using Sentence Transformers
- Indexes embeddings with FAISS
- Retrieves the most relevant passages based on semantic similarity

**Concepts learned**

- Embeddings
- Vector Search
- FAISS
- Cosine Similarity
- Document Chunking
